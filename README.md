## 📖 Description

The main goal of this project is to introduce me into the world of web development and improve my **HTML** layout skills and my ability to create responsive designs using **CSS/Tailwind**, as well as to gain an introduction to **JavaScript** and DOM manage.

## 🛠 Stack

- Vite
- HTML5
- Tailwind
- JavaScript

## 🚀 Installation


1. Clone this repository:

```bash
git clone https://github.com/Danilo-Doria/Danilo-Doria.github.io.git

cd Danilo-Doria.github.io.git
```

2. Excute scripts.

```bash
npm install
npm run dev
```
